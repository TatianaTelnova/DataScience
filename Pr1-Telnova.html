Выбор и описание данных¶

Выбраны данные Heart Disease с сайта https://archive.ics.uci.edu/ml/datasets/Heart+Disease. На их основе можно сделать прогнозирование заболеваний сердца.
На сайте представлены 4 базы данных по диагностике болезней сердца из следующих мед. учреждений:

  1. Фонд клиники Кливленда (cleveland.data)
  2. Венгерский институт кардиологии, Будапешт (hungarian.data)
  3. В.А. Медицинский центр, Лонг-Бич, Калифорния (long-beach-va.data)
  4. Университетская клиника, Цюрих, Швейцария (switzerland.data)
В каждой базе данных имеется информация о 76 показателях состояния пациента, на основе которых ставится диагноз. Присутствуют пропуски данных, отмечены значением -9,0.
Для анализа были взяты данные из Фонда клиники Кливленда (cleveland.data). Они содержат неинформативные столбцы с одинаковыми значениями во всех строках, не заполненные данными столбцы, а также столбцы, помеченные составителями базы данных как неиспользуемые. Такие столбцы не учитывались при дальнейшем анализе.

Описание показателей¶

Из 76 столбцов было выбрано 29, данные которых можно проанализировать.

Общая информация
1 id: уникальный номер пациента в списке
3 age: возраст на момент обследования (в годах)
4 sex: пол (1 = мужской; 0 = женский)

Данные истории болезни
9 cp: тип боли в груди
-- Значение 1: типичная стенокардия
-- Значение 2: атипичная стенокардия
-- Значение 3: неангинозная боль
-- Значение 4: бессимптомный
10 trestbps: систолическое артериальное давление в состоянии покоя при поступлении в больницу (в мм рт. ст.)
12 chol: содержание холестерина (в мг/дл)
16 fbs: уровень сахара в крови натощак > 120 мг/дл (1 = да; 0 = нет)
18 famhist: была ли у кого-то в семье ишемическая болезнь сердца (1 = да; 0 = нет)
19 restecg: результаты электрокардиографии в покое
-- Значение 0: нормальный
-- Значение 1: наличие аномалии ST-T (инверсия зубца T и/или элевация или депрессия ST > 0,05 мВ)
-- Значение 2: показывает вероятную или определенную гипертрофию левого желудочка по критериям Эстеса

Дата проведения ЭКГ
20 ekgmo: месяц проведения ЭКГ (от 1 до 12)
21 ekgday: день проведения ЭКГ (от 1 до 31)
22 ekgyr: год проведения ЭКГ (от 1981 до 1984)

Медикаментозное лечение в период проведения обследования
23 dig: прием дигиталиса в период проведения ЭКГ (1 = да; 0 = нет)
24 prop: прием бета блокаторов в период проведения ЭКГ (1 = да; 0 = нет)
25 nitr: прием нитратов в период проведения ЭКГ (1 = да; 0 = нет)
26 pro: прием блокаторов кальциевых каналов в период проведения ЭКГ (1 = да; 0 = нет)
27 diuretic: прием диуретиков в период проведения ЭКГ (1 = да; 0 = нет)

Данные электрокардиограммы
32 thalach: максимальная частота сердечных сокращений (кол-во раз)
33 thalrest: частота сердечных сокращений в покое (кол-во раз)
34 tpeakbps: пиковое артериальное давление при физической нагрузке, первая из 2 частей (в мм рт. ст.)
35 tpeakbpd: пиковое артериальное давление при физической нагрузке, вторая из 2 частей (в мм рт. ст.)
37 trestbpd: диастолическое артериальное давление в покое (в мм рт. ст.)
38 exang: стенокардия, вызванная физической нагрузкой (1 = да; 0 = нет)
39 xhypo: гипотензия, вызванная физической нагрузкой (1 = да; 0 = нет)
40 oldpeak: депрессия сегмента ST, вызванная физической нагрузкой, по сравнению с состоянием покоя (в мм)
41 slope: наклон пикового сегмента ST при нагрузке
-- Значение 1: восходящий
-- Значение 2: плоский
-- Значение 3: нисходящий

Данные рентгеноскопии сердца
44 ca: количество крупных сосудов, окрашенных при рентгеноскопии

Данные сцинтиграфии с таллием с физической нагрузкой
51 thal: результат сцинтиграфии с таллием
--Значение 3: нормальный
--Значение 6: фиксированный дефект
--Значение 7: обратимый дефект

Диагноз
58 num: наличие болезни сердца
-- Значение 0: нет
-- Значения 1-4: есть, предположительно, значение определяет степень тяжести заболевания
